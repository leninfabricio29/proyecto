INTRODUCCION

El desarrollo de software bajo políticas de desarrollo sustentable permite la construcción de proyectos informáticos ágiles, robustos, reusables y sobre todo que permitan el mejoramiento de actividades humanas. Sin embargo, la construcción de software requiere de una serie de metodologías y secuencia de actividades necesarias para asegurar la calidad del sistema, adicionalmente, la construcción del software requiere una planificación adecuada partiendo desde el establecimiento del alcance y qué objetivos se desea alcanzar. El presente informe, busca detallar la temática y los objetivos deseados durante la construcción de una plataforma web para el restaurante Carbonero Balcón ubicado en la ciudad de Loja. Además, se definen las metodologías, técnicas y arquitecturas que se implementarán como primer paso en la elaboración del sistema.

Desarrollo

2.1. Temática.

Plataforma WEB para la automatización de pedidos en el restaurante Carbonero Balcón de la ciudad de Loja.

2.2. Objetivos

2.2.1. Objetivo general

  • Diseñar una plataforma web para la automatización de los pedidos en el restaurante Carbonero Balcón en la ciudad de Loja bajo las normativa de desarrollo de software sustentable.

    • 2.2.2. Objetivos específicos
  • Analizar los procesos que se llevan a cabo en el restaurante referente a la recepción de pedidos. Conocer los conceptos relacionados a las metodologías de desarrollo de software sustentable.
  • Construir una solución informática para la automatización de pedidos en el restaurante utilizando una metodología ágil.
  • Implementar la solución informática en el tiempo establecido por el cliente.

    Lenguajes

    Los lenguajes de programación ayudan a los desarrolladores en la construcción de sistemas robustos, donde su implementación está netamente definida por el contexto y el sector de implementación. Sin embargo, primeramente se debe partir de la definición precisa sobre qué es un lenguaje de programación, de tal forma que, un lenguaje de programación, según Carlos Ureña Alamagro, se define como el conjunto de reglas y normas que permiten asociar cada programa con su fin objetivo y la forma en la que se va a desplegar [1]. Otra definición lo muestra como, un acuerdo o convenio de las reglas que se deben seguir para la construcción de programas informáticos [1]. En este contexto, se puede determinar que, un lenguaje de programación se puede definir como la representación formal de un conjunto de normas e instrucciones que permiten al desarrollador escribir una serie de órdenes, acciones, operaciones y algoritmos bajo el término de programa para controlar el funcionamiento de hardware y software [1][2]. Durante el presente proyecto se implementará una serie de lenguajes enfocados al desarrollo web, entre los que se va a implementar esta:

    Javascript

    Es un lenguaje de programación enfocado a la web, el cual permite implementar funciones de primera clase al ser un lenguaje ligero, interpretado y compilado bajo la filosofía de “just-in-time”. Es implementado en entornos como Nodejs, Apache, Adobe, entre otros. Otra definición, según Mozilla, es un lenguaje de programación basado en prototipos, multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y declarativa (por ejemplo programación funcional) [3]. En definitiva, el lenguaje permite el desarrollo ágil de aplicaciones enfocadas a la web basadas en la arquitectura de cliente-servidor tal es el caso de la plataforma del presente proyecto.

    Python

    Python es un lenguaje de programación de alto nivel que soporta múltiples implementaciones en el desarrollo de aplicaciones web, entre otros. En esencia, forma parte de la base del framework Django siendo el lenguaje que se implementa para el desarrollo de toda la aplicación completa. Otra de las definiciones que recibe python, según Python, se define como un lenguaje de programación de alto nivel orientado a objetos en compañía de una semántica agregada para el desarrollo web bajo la normativa de desarrollo rápido de aplicaciones o RAD [5][6]. Finalmente, es importante mencionar que, el uso del lenguaje viene orientado por el uso del framework de desarrollo Django, el cual permite el desarrollo de aplicaciones web ágiles

    2.5. Metodología, filosofía o arquitectura de software

    Metodología XP

    En sí el trabajo con una metodología permite disminuir el nivel dificultad, tener una organización de la tareas, agilizar procesos y mejorar el resultado final. Está metodología es un conjunto de técnicas, que evidencia principios como el desarrollo incremental. la participación activa del cliente, el interés en las personas y no en los procesos como elemento principal, y aceptar el cambio y la simplicidad. La principal ventaja de XP es que ayuda a potenciar las relaciones personales mediante el trabajo en equipo, impulsando la comunicación y eliminación de tiempos muertos.

  • Template
